## ProprietaryNetwork
Unityでオンラインゲーム制作を支援するライブラリのプロトタイプです。

## 概要
- サーバーとクライアントの通信を簡単に行うことができます。
- サーバーサイドはピュアなC#で書けるようにすることで、Unity以外の環境でも利用できるようにします。
- オンラインゲームのネットワーク構造を理解することを目的に作成しています。
